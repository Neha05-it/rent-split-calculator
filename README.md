## How to Run
1. Install Python
2. Save the file as rent_calculator.py
3. Run the program:
   python rent_calculator.py

## Example
Enter your hostel/flat rent = 8000  
Enter the amount of food ordered = 2000  
Enter the total electricity spend = 1000  
Enter the number of persons living in room/flat = 4  

Each person will pay = 2750
